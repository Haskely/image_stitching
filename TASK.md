# Task

写一个拼长图功能的 html，要求功能：

1. 支持多张图片竖向拼接
2. 每张图片都支持独立竖向裁切（面向多张截图竖向拼接场景）
3. 希望支持拖拽式裁剪，然后所见即所得的那种交互
4. 触摸屏友好
4. 支持部署到 github.io
